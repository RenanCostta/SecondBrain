Usamos o Loop **while** quando a condição de parada não é baseada em um número fixo de iterações.

//Código exemplo

int i = 0;
while (i < 5) {
    System.out.println("Iteração: " + i);
    i++;
}

- Executa enquanto `i < 5` for verdadeiro.
